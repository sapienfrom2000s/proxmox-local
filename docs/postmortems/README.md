# Postmortems

This folder contains postmortem documents for incidents, outages, and other
significant issues.

Each postmortem follows a structured format:

- **Incident Summary**: What happened and when
- **Root Cause**: The underlying reason for the incident
- **Timeline**: Key events and actions taken
- **Resolution**: How the issue was resolved
- **Action Items**: Follow-up tasks to prevent recurrence

## File Naming

Use the format `YYYY-MM-DD-topic.md` for new postmortems.
